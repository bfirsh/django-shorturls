Change log
==========

3.0.0 (2026-03-01)
------------------

- Require Python 3.10+.
- Add support for Django 4.2 and 5.2.
- Drop support for all Django versions older than 4.2.
- Drop the ``six`` dependency; use stdlib and modern Django imports.
- Replace ``setup.py`` with ``pyproject.toml``.
- Replace Travis CI with GitHub Actions.

2.0.0 (2016-12-15)
------------------

- Add support for Python 3.
- Add support for Django 1.9 and 1.10.
- Drop support for Python 2.6.
- Drop support for Django 1.4, 1.5, 1.6, and 1.7.

1.1.1 (2015-09-29)
------------------

- Added support for Django 1.8

1.1.0 (2014-10-31)
------------------

- Added support for Django 1.7
- Added a Crockford base 32 encoder
- Made base conversion significantly faster

1.0.1 (2009-04-13)
------------------

Initial release.
