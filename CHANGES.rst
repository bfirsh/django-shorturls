Change log
==========

3.0.0 (2026-02-28)
------------------

- Add support for Django 4.2, 5.0, 5.1, and 5.2.
- Add support for Python 3.8 through 3.13.
- Drop support for Python 2.7, 3.4, 3.5, and 3.6.
- Drop support for Django 1.8, 1.9, and 1.10.
- Remove dependency on ``six``.

2.0.0 (2016-12-15)
------------------

- Add support for Python 3.
- Add support for Django 1.9 and 1.10.
- Drop support for Python 2.6.
- Drop support for Django 1.4, 1.5, 1.6, and 1.7.

1.1.1 (2015-09-29)
------------------

- Added support for Django 1.8

1.1.0 (2014-10-31)
------------------

- Added support for Django 1.7
- Added a Crockford base 32 encoder
- Made base conversion significantly faster

1.0.1 (2009-04-13)
------------------

Initial release.
